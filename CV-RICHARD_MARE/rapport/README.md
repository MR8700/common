# Page CV – Richard MARE

## Description
Page personnelle de Richard MARE dans le cadre du projet collaboratif de création d’un site web statique regroupant les CV des participants.

## Contenu
- **À propos de moi** : informations personnelles et ambition.
- **Compétences** : C, JavaScript, PHP, Laravel, React Native, MySQL, Git/GitHub, API REST.
- **Formations** : plusieurs formations académiques et pratiques.
- **Projets clés** : E-commerce sécurisé, application mobile étudiants, portfolio pro.
- **Contact** : téléphone, email, GitHub, Facebook, localisation.

## Organisation
