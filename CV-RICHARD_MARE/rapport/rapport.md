# Rapport de projet – Page CV de Richard MARE

## Introduction
Dans le cadre du projet collaboratif de création d’un site web statique regroupant les CV des participants, j’ai développé ma page personnelle afin de présenter mon profil, mes compétences, mes formations et mes projets clés.

## Objectifs
- Créer une page CV moderne et responsive.
- Mettre en pratique HTML, CSS et JavaScript (accordéons, carrousels, effets typing).
- Travailler en collaboration via Git/GitHub.

## Déroulement

### 1. Création de ma page
- Développement d’une page HTML avec structure claire et sections : À propos, Compétences, Formations, Projets, Contact.
- Mise en page avec CSS responsive.
- Ajout d’interactions JavaScript : accordéons, carrousels, texte tournant autour du drapeau, effet typing pour l’état civil.

### 2. Intégration dans le dépôt commun
- Création du dossier `/participants/richard/` dans le dépôt principal.
- Upload des fichiers : `index.html`, `style.css`, images.
- Pull request pour validation et intégration.

## Contenu de la page
- **À propos** : informations personnelles et ambition.
- **Compétences** : C, JavaScript, PHP, Laravel, React Native, MySQL, Git/GitHub, API REST.
- **Formations** : Licence Informatique Appliquée, Rédaction de projet percutant, Gestion de secrétariat public, Musique/chant/batterie, Conduite de groupe et travail d’équipe, Mentorat politique des jeunes.
- **Projets clés** : E-commerce sécurisé, Application mobile étudiants, Portfolio professionnel.
- **Contact** : téléphone, email, GitHub, Facebook, localisation.

## Technologies utilisées
- HTML5, CSS3 (responsive)
- JavaScript (interactions)
- Git/GitHub pour versioning et collaboration
- Font Awesome pour icônes

## Conclusion
Ma page CV est fonctionnelle, visuellement agréable, et entièrement intégrée dans le site collaboratif. Elle reflète mes compétences, expériences et ambitions professionnelles.
