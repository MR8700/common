# Cahier des charges – Page CV de Richard MARE

## Objectif
Créer une page CV personnelle claire, moderne et responsive, présentant les informations personnelles, compétences, formations, projets et contacts de Richard MARE.

## Fonctionnalités
- **À propos de moi** : nom, prénom, âge, statut, résidence, études, ambition.
- **Compétences** : badges stylisés pour chaque compétence.
- **Formations** : carrousel présentant les différentes formations suivies.
- **Projets clés** : carrousel de projets réalisés.
- **Contact** : cartes avec téléphone, email, GitHub, Facebook, localisation.
- Interactions JavaScript : accordéons pour sections, carrousels, texte tournant autour du drapeau, effet typing pour l’état civil.

## Contraintes
- Mise en page responsive pour mobile, tablette et desktop.
- Respect des bonnes pratiques HTML/CSS/JS.
- Intégration dans le dépôt collaboratif via pull request.
- Respect de la date limite : samedi 10 mai 2025.

## Livrables
- `index.html` – page CV personnelle
- `script.js` – pour les traitements et le dynamisme de la page
- `style.css` – styles de la page
- `images/` – photos et logos
- Rapport et README personnels
